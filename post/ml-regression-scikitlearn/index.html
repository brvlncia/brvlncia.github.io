Machine Learning Regression Implementation with SciKit Learn

March 8, 2022 · 1 min · Bryan Valencia

ScitKitLearn Simple Linear Regression Implementation with Sample Dataset

I load in the necessary libraries.


import pandas as pd
from sklearn import model_selection, linear_model

I read my dataset in with pandas and define my features.

df = pd.read_csv('dataset.csv')
features = ['col0','col1','col2','col3','col4','col5','col6','col7','col8','col9','col10']

I define my model and set an X and y variable to set up cross-validation.

model = linear_model.LinearRegression()
X = df[features]
y = df.label

xval = model_selection.KFold(10, shuffle=True)
results = model_selection.cross_validate(model, X, y, cv=xval, scoring='r2', return_train_score=True)

I calculate mean R^2 and standard deviation.

train_r2 = results['train_score'].mean()
train_r2_sd = results['train_score'].std()
test_r2 = results['test_score'].mean()
test_r2_sd = results['test_score'].std()
